MinecraftAdmin is a plugins for display status of a Minecraft server. 

== Description ==

MinecraftAdmin is for manage and display information of Minecraft server.

This plugins work with Minecraft server craftbukkit : http://www.bukkit.org.
The plugins server use is :

* Whitelist : http://forums.bukkit.org/threads/admn-gen-whitelist-v2-7-431-556.587/
* Remote Toolkit : http://forums.bukkit.org/threads/admn-remotetoolkit-r10-a8-3-restarts-advanced-backup-preview-full-remote-console-531.674/
* TCP Bukkit : http://forums.bukkit.org/threads/admin-dev-tcp-interface-for-bukkit-v1-2-493.746/

Feature :

* Manage user with Whitelist
* Display status of manager RemoteToolkit
* Display status of server
* Display user connected
