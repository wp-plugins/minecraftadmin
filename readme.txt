MinecraftAdmin is a plugins for display status of a Minecraft server. 

== Description ==

MinecraftAdmin is for manage and display information of Minecraft server.

This plugins work with Minecraft server craftbukkit : http://www.bukkit.org.
The plugins server use is :

* Whitelist : http://forums.bukkit.org/threads/587/
* Remote Toolkit : http://forums.bukkit.org/threads/674/
* TCP Bukkit : http://forums.bukkit.org/threads/746/
* Permissions : http://forums.bukki.org/threads/5974/

Feature :

* Manage user with Whitelist
* Display status of manager RemoteToolkit (Widget)
* Display status of server (Widget)
* Display user connected (Widget)
* Manage permissions by groups with Permissions

Please backup your permissions files before using the modules.
